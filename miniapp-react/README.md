# TON Consensus Mini App

React + Vite Telegram Mini App for TON Consensus P2P betting platform.

## Features
- Create and join bets
- TON Connect wallet integration
- Real-time bet status updates
- Oracle dispute visualization
- Share bet invite links

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

Output goes to `dist/` folder served by Express backend.
